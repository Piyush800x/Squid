# Fast Squid Proxy Installer with Basic Authentication UBUNTU
## Clone this repo
```
git clone https://github.com/Piyush800x/Squid.git
```
## Change directory
```
cd Squid
```
## Run main.py
```
sudo python3 main.py
```
## Add IP routes
```
wget https://raw.githubusercontent.com/Piyush800x/Squid/master/ips.py
```
## Run it
```
sudo python3 ips.py
```
